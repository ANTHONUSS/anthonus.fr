J'aime tout dans l'informatique, Commençant par le développement web / applications, aux petits programmes marrants, passant par des bots Discord, des petits jeux vidéos ou même de l'IA, jusqu'aux configurations et installations de posts informatique, autant Windows que Linux. Mes projets

  • ANTHONUS-Bot

    ANTHONUS-bot est un bot discord que j'ai créé pour mes serveurs Discord.
    Son but n'est pas d'être rendu public à l'usage, puisqu'il a principalement été créé pour un usage personnel et pour mes amis.
    Il possède des commandes fun et troll, une intégration d'IA, une fonction de téléchargement de vidéo youtubes ainsi qu'une gestion complète de musiques dans les salons vocaux.
    Il a été fait en Java avec la bibliothèque JDA, les API de Google, Youtube, Reddit et OpenAI.

  • Noob-Bot

    Noob-Bot est un bot discord créé pour mon Serveur Discord, Noob Nation.
    Il possède des fonctionnalités de modération, d'accueil des nouveaux utilisateurs, ainsi qu'un système de niveaux et de récompenses.
    Il a été fait en Java avec la bibliothèque JDA, et utilise une base de données SQLite pour stocker les données des utilisateurs et des serveurs.

  • Custom Assistant

    Custom assistant est mon plus gros projet à ce jour.
    Il s'agit d'une application de bureau qui permet d'avoir un assistant vocal personnalisable.
    Il fonctionne avec ChatGPT et permet de lui donner une personnalité, lui donner une voix personnalisée, recherches web, etc.
    Il est fait en Java, avec les api nécéssaires pour son fonctionnement en vocal et communication avec ChatGPT.

  • HappyVirus

    HappyVirus n'a rien d'un virus, mais est une application en Java qui affiche à un intervalle de temps des images aléatoires de memes.
    Ce projet a purement été créé pour tester les fenêtres, images, son et gifs en swing et n'a rien de dangereux.

  • FindTheLuigi

    Ce projet est un mini-projet de jeu fun à rapidement faire en Java.
    Il s'agit d'une copie simplifiée du mini jeu de New Super Mario Bros. sur Nintendo DS où il faut trouver un certain personnage parmi les autres.
    Le but de cette application est de trouver le Luigi caché parmi les autres personnages du jeu.
    Il a été fait en Java avec Swing pour l'interface graphique.

  • LOGs

    LOGs est une library Java qui permet de gérer de façon plus simple les logs dans un programme.
    Elle permet de gérer les logs dans un fichier texte, des couleurs, des catégories et le temps exact de l'envoi du log.

  • SymLinkCreator

    SymLinkCreator est une application Java qui permet de créer des liens symboliques sur Windows facilement.
    Il suffit de sélectionner le dossier ou fichier source ainsi que la cible, et le programme créera un lien symbolique entre ces deux-ci.
    Il a été fait en Java avec JavaFX pour l'interface graphique.

  • DiscordBackground

    Petit theme Discord en CSS qui fonctionne avec Vencord (et très probablement d'autres modifications de Discord) qui permet de rajouter un fond d'écran aux discussions.
    L'image est chargée depuis un URL, et peut être modifiée à tout moment, avec l'opacité et le flou de l'image.
