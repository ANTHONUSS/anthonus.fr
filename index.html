Mes projets

  • ANTHONUS-bot

    ANTHONUS-bot est un bot discord que j'ai créé pour mes serveurs Discord.
    Son but n'est pas d'être rendu public à l'usage, puisqu'il a principalement été créé pour troll et pouvoir s'amuser avec mes amis.
    Il possède des commandes fun et troll, mais également de téléchargement et gestion complète de musiques, file d'attente, etc.
    Il a été fait en Java avec l'API JDA.

  • JChatBot

    Useful bot est un bot Discord créé pour but de faire quelques actions utiles pour les utilisateurs.
    Il est prévu pour un usage sur serveur, ou juste sur un serveur personnel pour une utilisation personnelle.
    Actuellement, il possède deux commandes principales : Une pour pouvoir télécharger des videos Youtube (en tant que musique, ou vidéo), et une pour prévenir des sorties d'animés récupérées depuis votre compte AniList.
    Il est fait en Java avec l'API JDA, utilise YT-dlp pour télécharger les vidéos Youtube, et l'API d'AniList pour les animés.

  • Useful Bot

    JChatBot est un bot Discord créé pour pouvoir faire des conversations "humaines" avec les utilisateurs.
    La personne qui héberge le bot peut lui donner la personnalité qu'il veut, et le bot répondra en fonction de cette personnalité.
    Le bot possède aussi un système qui lui permet de se souvenir des discussions passées.
    Il a été fait en Java avec l'API JDA et communique avec ChatGPT via l'API d'OpenAI.

  • Custom Assistant

    Custom assistant est mon plus gros projet à ce jour.
    Il s'agit d'une application de bureau qui permet d'avoir un assistant vocal personnalisable.
    Il fonctionne avec ChatGPT et permet de lui donner une personnalité, lui donner une voix personnalisée, recherches web, etc.
    Il est fait en Java, avec les api nécéssaires pour son fonctionnement en vocal et communication avec ChatGPT.

  • HappyVirus

    HappyVirus n'a rien d'un virus, mais est une application en Java qui affiche à un intervalle de temps des images aléatoires de memes.
    Ce projet a purement été créé pour tester les fenêtres, images, son et gifs en swing et n'a rien de dangereux.

  • LOGs

    LOGs est une library Java qui permet de gérer de façon plus simple les logs dans un programme.
    Elle permet de gérer les logs dans un fichier texte, des couleurs, des catégories et le temps exact de l'envoi du log.

  • SymLinkCreator

    SymLinkCreator est une application Java qui permet de créer des liens symboliques sur Windows facilement.
    Il suffit de sélectionner le dossier ou fichier source ainsi que la cible, et le programme créera un lien symbolique entre ces deux-ci.
    Il a été fait en Java avec JavaFX pour l'interface graphique.

  • DiscordBackground

    Petit theme Discord en CSS qui fonctionne avec Vencord (et très probablement d'autres modifications de Discord) qui permet de rajouter un fond d'écran aux discussions.
    L'image est chargée depuis un URL, et peut être modifiée à tout moment, avec l'opacité et le flou de l'image.
